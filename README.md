# Crypto Coin Tracker

This is a simple web app for displaying current information about top crypto coins.

Created with
- [Vite](https://vitejs.dev)
- [Pico.css](https://picocss.com/)
- SCSS
- Axios
- [Coinstats API](https://cryptocointracker.com/api/coinstats)
